DSA Topics

Linked List

A linked list is a data structure composed of nodes, each holding data and a reference to the next node. It forms a dynamic sequence where the last node points to null.

Stack

A stack is a fundamental data structure that follows the Last In, First Out (LIFO) principle. It operates like a collection of items where new elements are added to the top, and only the topmost element can be removed or accessed at any given time.

Queue

A queue is a basic data structure that adheres to the First In, First Out (FIFO) principle. It functions as a collection of elements where new elements are added at the rear, and removal or processing occurs from the front.

Tree

A tree is a hierarchical data structure composed of nodes, where each node holds data and can have zero or more child nodes. The topmost node is called the root, and nodes with no children are leaves. Nodes between the root and leaves are internal nodes.

Sorting

Sorting is the process of arranging elements in a specific order, often ascending or descending, based on a defined criterion. It is a fundamental operation in computer science and plays a crucial role in various applications.

Searching

Searching is the process of finding a specific item or element in a collection of data. Various searching algorithms are employed based on the structure of the data and the requirements of the search.

Graph

A graph is a fundamental data structure that consists of a set of nodes (vertices) and a set of edges connecting these nodes. Graphs are used to represent relationships between entities and are employed in various applications, including social networks, transportation systems, and computer networks.
